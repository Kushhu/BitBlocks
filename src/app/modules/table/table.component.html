Table

• <bit-table>

Allows you to define table content directly in your template using <bit-table-row> and <bit-table-cell> elements. This approach is useful for simple tables with static data.

{{snippet.basic}} AX902 Electric Bike 30,000 AX903 Electric Car 2,25,000

Basic Data Table

• <bit-table>

Data Table allows list of entities togather for complex data handling.

  • Supports dynamic data binding using the [data] input.
  • Automatically generates columns based on object keys if columns are not specified.
  • Ideal for displaying lists of entities such as products, users, or orders.
  • Enables sorting, filtering, and pagination (if implemented in your component).

Table Into Card View

• bitCard

The bitCard template allows you to display each row of your data as a card instead of a traditional table row. Use the let-row variable to access each data item within the card template. This is useful for responsive layouts or when you want a more visual presentation of your data.

  • Define a ng-template with the #bitCard reference inside your <bit-table> component.
  • Use let-row to access the current row's data.
  • Customize the card layout as needed using HTML and Angular bindings.
