Table

• <bit-table>

A basic code defined approch to declare table content.

The <bit-table> component allows you to define table content directly in your template using <bit-table-row> and <bit-table-cell> elements. This approach is useful for simple tables with static data.

{{snippet.basic}} 9 Kushagra 30 8 Ramni 22

Basic Data Table

• <bit-table>

Data Table allows list of entities togather for complex data handling.

  • Supports dynamic data binding using the [data] input.
  • Automatically generates columns based on object keys if columns are not specified.
  • Ideal for displaying lists of entities such as products, users, or orders.
  • Enables sorting, filtering, and pagination (if implemented in your component).
